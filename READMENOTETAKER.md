# NoteTaker

Created an app to take notes from a user and save them.

User will enter the site by visiting the landing page.

From there the user can progress by pressing the "Get Started" button.

User will input a note title and a note body message.

User can save the message by pressing the save icon that appears at the top right corner.

User will see their note appear to the left when that is saved.

User can delete the note by pressing the trash icon to the right of their notes.
